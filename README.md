# CampusClub - University Student Hub

## 🎓 Project Overview

**CampusClub** is a comprehensive web-based platform designed to enhance the university student experience by centralizing essential academic and social resources. Developed as part of the ITCS333 course using **Replit** as our collaborative development environment, this project brings together multiple features to help students navigate campus life more effectively.

## ✨ Core Features

### 📅 Events Calendar
- Browse upcoming campus events
- Filter events by date, category, or department
- View event details and descriptions

### 👥 Study Group Finder
- Search for study groups by course or subject
- Join existing groups or create new ones
- Connect with peers for collaborative learning

### ⭐ Course Reviews
- Read and write reviews for university courses
- Share insights about course difficulty, workload, and professors
- Help fellow students make informed course selections

### 📚 Course Notes
- Upload and download course notes
- Share study materials with classmates
- Access a growing repository of academic resources

### 🎪 Club Activities
- Discover student club events and meetings
- Browse club profiles and membership information
- Stay updated on extracurricular opportunities

## 📂 Project Structure

The repository is organized by feature modules:
- **`Calnder-moduel/`** - Events calendar functionality
- **`study-group-finder/`** - Study group search and management
- **`CourseReviews/`** - Course review system
- **`Course Notes/`** - Note sharing feature
- **`Club Activites/`** - Club activities display
- **`images/`** - Shared image assets
- **`index.html`** - Main entry point
